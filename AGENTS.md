# 加密工具 - 项目概述

## 目录用途
这是一个简单易用的AES加密/解密工具集合，主要用于：
- 加密/解密文本字符串（用于聊天保密）
- 加密/解密文件（照片、文档、视频等）

## 目录结构
```
H:\加密工具\
├── 加密工具\               # 工具主目录
│   ├── 加密.exe          # 加密程序
│   ├── 解密.exe          # 解密程序
│   └── 使用说明.txt      # 详细使用说明
└── AGENTS.md             # 本文件
```

## 关键文件说明

### 加密.exe
- 功能：执行加密操作
- 支持两种模式：
  1. 字符串加密：加密文本内容
  2. 文件加密：加密文件内容，生成 .aes 文件

### 解密.exe
- 功能：执行解密操作
- 支持两种模式：
  1. 字符串解密：解密加密的文本
  2. 文件解密：解密 .aes 文件，还原原始文件

### 使用说明.txt
- 包含详细的操作步骤
- 提供使用示例和重要提醒

## 使用方法

### 字符串加密/解密流程
1. **加密**：
   - 双击 `加密.exe`
   - 选择选项 1（字符串加密）
   - 输入密钥（例如：123456）
   - 输入要加密的文字
   - 复制生成的密文发送

2. **解密**：
   - 双击 `解密.exe`
   - 选择选项 1（字符串解密）
   - 输入与加密时相同的密钥
   - 粘贴密文
   - 查看解密后的明文

### 文件加密/解密流程
1. **加密文件**：
   - 将待加密文件放入工具目录
   - 双击 `加密.exe`
   - 选择选项 2（文件加密）
   - 输入密钥
   - 输入文件名（例如：照片.jpg）
   - 生成 `encrypted.aes` 文件

2. **解密文件**：
   - 将 `encrypted.aes` 文件放入工具目录
   - 双击 `解密.exe`
   - 选择选项 2（文件解密）
   - 输入与加密时相同的密钥
   - 输入 `encrypted.aes`
   - 输入解密后的文件名（例如：照片_解密.jpg）
   - 还原原始文件

## 重要提醒
1. **密钥管理**：密钥必须妥善保存，丢失密钥将导致无法解密
2. **密钥一致性**：解密时使用的密钥必须与加密时完全一致
3. **离线使用**：工具无需安装和联网，完全离线运行
4. **安全性**：建议使用强密钥以提高安全性

## 技术特性
- 加密算法：AES（高级加密标准）
- 操作模式：离线执行
- 支持文件类型：所有文件类型（照片、文档、视频等）
- 输出格式：加密文件为 .aes 扩展名

## 适用场景
- 个人隐私保护
- 敏感信息传输
- 文件安全存储
- 即时通讯加密
